# 🟢 Python CLI Nutrition & Macro Tracker

A lightweight, Object-Oriented Command Line Interface (CLI) application built in Python to track daily nutritional intake. 

This project demonstrates core software engineering principles for data pipelines, including OOP class structure, functional data aggregation, interactive user loops, and persistent file-based state management.

## ✨ Features
* **Multi-Dimensional Tracking:** Logs Calories, Protein, Carbohydrates, and Fats.
* **Persistent Storage:** Automatically serializes data state to a local `.json` file, ensuring zero data loss between sessions.
* **Interactive CLI Menu:** Robust `while` loop interface with `try/except` error handling to prevent application crashes from invalid user input.
* **Aggregated Reporting:** Functional list comprehensions calculate daily totals and track progress against hardcoded targets.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Standard Libraries:** `json`, `os`, `typing`
* **Architecture:** Object-Oriented Programming (OOP) & Functional logic
